ماذا يحتوي كل ملف؟

📊 ملف CSV يحتوي على:

  • معرف التصنيف وتاريخ الطلب
  • عنوان التذكرة ونصها
  • النموذج المستخدم
  • نتائج L1 / L2 / L3 مع نسبة الثقة
  • وقت API ووقت الاستدلال
  • تقييم المستخدم (إعجاب / عدم إعجاب، تعليق، بريد)

🗄️ ملف SQL يحتوي على:

  • جميع الجداول (التصنيفات، التقييمات، سجل المراقبة، الزوار)
  • البيانات والفهارس والمخططات
  • صالح للاستعادة بـ: sqlite3 itsm_restored.db < itsm_backup.sql
